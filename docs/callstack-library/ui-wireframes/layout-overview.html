📚 콜스택 도서관 - 레이아웃 설계 총괄

난이도별 차별화된 학습 경험을 위한 레이아웃 시스템

🆕 최신 버전

📋 주요 개선 사항

  • 초급 (1-8): 함수 호출 순서만 예측 (기존 방식 유지)
  • 중급 (9-16): 함수 시작/종료 모두 예측하여 LIFO 원칙 학습
  • 고급 (17-24): 각 실행 시점의 스택 상태를 직접 구성
  • 난이도별 전용 레이아웃으로 점진적 학습 곡선 구현
